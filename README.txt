ACF: Customizer enable you to create the custom theme options by ACF. Live result change available

== Description ==

ACF: Customizer take advantage of Advanced Custom Field to create the theme options, it's useful for freelancer or plugin/theme developer, when you want to improve your product experience.

ACF: Customizer at the moment is under development and need improve a lot. So you can use it at your own risk.

Right now, ACF: Customizer is compatible with ACF's default fields (both FREE & PRO - repeater field too). We will extend it in the future to compatible with other field types in the future.

Also, ACF: Customizer is made to work with WordPress native customizer, so you can use WP Customizer's functions to display the value as well as live change when user do the customization.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `acf-customizer` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create your field group at ACF menu and set the field group rule to Customizer.
